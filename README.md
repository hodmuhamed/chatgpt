# Humperia Homepage Concept

A concept Next.js homepage for [Humperia.shop](https://humperia.shop) showcasing a modern, editorial shopping experience that can be powered by a WordPress + WooCommerce backend.

## Getting started

```bash
npm install
npm run dev
```

The development server runs at [http://localhost:3000](http://localhost:3000).

## Tech stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS 3
- TypeScript
